Présentation

MemoryCloud est un projet personnel, une application fullstack de type SaaS permettant le stockage, la gestion et l’organisation d’images dans une interface moderne et intuitive.


Le projet repose sur une architecture découplée avec un frontend en React et un backend en Laravel exposant une API REST.


Le backend gère l’ensemble de la logique métier : authentification des utilisateurs, gestion des fichiers, stockage des métadonnées et sécurisation des accès via un système d’authentification basé sur cookies CSRF (Laravel Sanctum).


Le frontend React consomme l’API pour offrir une expérience utilisateur fluide, avec gestion des sessions, affichage dynamique des images et interactions en temps réel.


Le projet utilise Docker en environnement de développement pour simplifier la configuration et l’exécution de l’application. Nginx est utilisé comme serveur web pour gérer les requêtes et servir l’application.


Ce projet m’a permis de renforcer mes compétences en développement fullstack, mais également d’explorer des notions proches de l’environnement DevOps, notamment à travers la conteneurisation avec Docker, la gestion d’un serveur web avec Nginx et la structuration d’une application orientée production.


Le projet constitue une base évolutive sur laquelle je prévois d’intégrer des pratiques DevOps plus avancées, notamment l’automatisation des déploiements via CI/CD et une approche plus structurée de mise en production.
